# DCIT 308 Lectures 2 and 3 Individual Assignment Package

This package contains:
- Assignment brief
- Submission checklist and cover sheet
- Java starter code with clear TODOs
- Sample and template CSV files
- Pseudocode support file
- Expected output shape
- Clean diagrams used in the brief

Students must localise the assignment to a Ghanaian service context and use their own index-number parameters.
